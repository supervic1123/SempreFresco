Politica de Privacidade - Sempre Fresco

Ultima atualizacao: 14/04/2026

Esta Politica de Privacidade explica como Sempre Fresco acessa, usa, armazena e protege informacoes relacionadas ao uso do aplicativo movel.

1. Controlador e contato

Aplicativo: Sempre Fresco

Contato de privacidade e suporte: vikena.smart@gmail.com

2. Dados tratados pelo app

O Sempre Fresco foi projetado para funcionar principalmente com dados armazenados no dispositivo do usuario. O app pode tratar as seguintes categorias:

  • Dados inseridos pelo usuario: nome do produto, marca, descricao, categoria, localizacao, quantidade, preco, codigo de barras ou QR e data de validade.
  • Dados tecnicos locais: idioma selecionado, horario do lembrete, status das notificacoes e caminhos de pastas escolhidas pelo usuario para backups.
  • Dados relacionados a camera: leitura de codigos de barras e codigos QR quando o usuario inicia voluntariamente o scanner.
  • Arquivos de backup: arquivos JSON exportados ou restaurados pelo usuario com informacoes do inventario e produtos lembrados.

3. Permissoes e finalidade

  • Camera: usada apenas para escanear codigos de barras e QR quando o usuario solicitar.
  • Notificacoes: usadas para enviar lembretes locais sobre produtos proximos do vencimento, somente se o usuario ativar a funcao.
  • Acesso a arquivos selecionados pelo usuario: usado para importar ou exportar backups manualmente por meio do seletor de arquivos ou pasta.
  • Internet: usada para abrir links legais externos e, se o usuario usar recursos que dependam de servicos externos configurados pelo desenvolvedor, para realizar requisicoes HTTPS.

4. Como os dados sao usados

  • Exibir e organizar o inventario de alimentos.
  • Detectar produtos proximos da validade e gerar lembretes locais.
  • Lembrar informacoes associadas a codigos escaneados para agilizar futuras entradas.
  • Criar e restaurar backups locais solicitados pelo usuario.
  • Aplicar preferencias de idioma e configuracoes dentro do app.

5. Armazenamento e retencao

As informacoes do inventario e as configuracoes sao armazenadas localmente no dispositivo do usuario. Os backups automaticos e manuais permanecem disponiveis ate que o usuario os exclua, desinstale o app ou os sobrescreva.

6. Compartilhamento de dados

O Sempre Fresco nao vende dados pessoais. O app nao compartilha automaticamente informacoes do inventario com terceiros para fins publicitarios.

Pode haver transferencia de dados apenas nestas situacoes:

  • Quando o usuario exporta ou compartilha voluntariamente um arquivo de backup.
  • Quando o usuario abre um link externo.
  • Quando um recurso opcional configurado pelo desenvolvedor usa um servico externo de traducao ou de politica de versao por HTTPS.

7. Servicos de terceiros

O aplicativo pode integrar bibliotecas ou servicos tecnicos necessarios para funcionalidades locais do dispositivo, como notificacoes, selecao de arquivos, compartilhamento de arquivos ou leitura de codigos. Se um servico externo de traducao ou de politica de versao for configurado, o texto enviado a esse servico sera processado de acordo com os termos e praticas de privacidade do respectivo fornecedor.

8. Seguranca

Medidas razoaveis sao adotadas para proteger as informacoes tratadas pelo app. No entanto, nenhum metodo de armazenamento ou transmissao e totalmente infalivel, e o usuario deve proteger o acesso ao seu dispositivo e aos seus arquivos de backup.

9. Criancas

O app nao e direcionado especificamente a criancas e nao busca coletar conscientemente dados pessoais de criancas.

10. Controles do usuario

  • O usuario pode editar ou excluir itens do inventario dentro do app.
  • O usuario pode desativar notificacoes pelo app ou pelas configuracoes do dispositivo.
  • O usuario pode excluir arquivos de backup criados manual ou automaticamente.
  • O usuario pode desinstalar o app para remover o armazenamento local gerenciado pelo aplicativo, exceto arquivos exportados para fora da area privada do app.

11. Alteracoes nesta politica

Esta Politica de Privacidade pode ser atualizada para refletir mudancas legais, tecnicas ou funcionais. A versao vigente sera sempre a publicada nesta pagina.

12. Contato
